# sof_es6_pm

Code samples for Stackoverflow question about process managers for ES6 modules.

# Build app
```
$ npm install
```



# CommonJS app

## Start app using nodejs
```
$ node ./src/app_cjs.js
```

## Start app using process manager `pm2`
```
$ node ./src/app_cjs.js
```



# ES6 app

## Start app using nodejs
```
$ node --experimental-modules ./src/app_es6.mjs
```
